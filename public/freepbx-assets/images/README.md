# FreePBX Custom Theme Images - HRTelecoms

Placez vos images personnalisées ici:

## Images requises:

1. **sys-admin.png** - Icône pour "Administration" et "User Control Panel" sur la page de login
   - Dimensions recommandées: 160x160px
   - Format: PNG avec transparence

2. **user-control.png** - Icône pour "User Control Panel"
   - Dimensions recommandées: 160x160px
   - Format: PNG avec transparence

3. **operator-panel.png** - Icône pour "Operator Panel"
   - Dimensions recommandées: 160x160px
   - Format: PNG avec transparence

4. **support.png** - Icône pour "Get Support"
   - Dimensions recommandées: 160x160px
   - Format: PNG avec transparence

## Utilisation:

Ces images seront automatiquement servies par Laravel et injectées dans FreePBX via le CentrexProxyController.

Les URLs seront:
- http://votre-domaine.com/freepbx-assets/images/sys-admin.png
- http://votre-domaine.com/freepbx-assets/images/user-control.png
- http://votre-domaine.com/freepbx-assets/images/operator-panel.png
- http://votre-domaine.com/freepbx-assets/images/support.png

## Branding recommandé:

- Utilisez les couleurs HRTelecoms (#5865F2 - bleu principal)
- Style moderne et épuré
- Icônes avec transparence pour s'adapter au thème
